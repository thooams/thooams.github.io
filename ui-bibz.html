Ui Bibz

Bootstrap Framework with Ruby on Rails

One component to rule them all

Library Ui Bibz is based on one component. All components inherit from component element. No need to write HTML, CSS or Javascript. Component are based on Bootstrap and all new version are manage by Ui Bibz. The framework is writted with Ruby for Ruby on Rails.

A component will write always the same way even if HTML, CSS or Javascript code change. You can find all components of Bootstrap 4 and more.

Ui

Core Framework

Alerts, inputs, breadcrumbs, cards, cols, buttons, dropdowns and +30 components are availabled right now.

Content, options, html_options

Ui Bibz component is based on Ruby on Rails link_to method with some specificity. 

                      
def component content = nil, options = nil, html_options = nil, &block
end
The content can be insert inline on content variable or by block. Block can also be used to configure element present inside the component (tap method). Options and html_options are hashes present to configure the component or his html.

That 's All !

All component are based on this fundamental principe. Some element are variant but copy on the same model.

Technology: Docker / Ruby on Rails / Coffescript and Extjs / HTML and CSS

Simple and efficient

Library is organize by namespace like UiBibz::Ui::Buttons::ButtonLink and for facilitate and improve use of Ui Bibz, helpers are created for call render of component very easily with simple methods: cols, dropdown, button_link...

All input components are compatible with simple_form to create forms very easily.

Tested, Gemified, Installed

All Components are tested with units tests (Minitest). Library follow latest version of Ruby on Rails, Bootstrap or FontAwesome and updated monthly.

Library is downloadable as gem on Rubygem or as source code on Github.

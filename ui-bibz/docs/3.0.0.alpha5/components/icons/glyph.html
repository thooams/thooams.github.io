Glyph(ui_glyph)

UiBibz::Ui::Core::Icons::Glyph

Icon component includes over 634 glyphs in font format from the fontawesome.io set (version 5.15.1).

Usage

This component is an extension of component element. A Ui Bibz component consists of 3 arguments:
  • content[value/block]
  • options[hash]<default: {}>
  • html_options[hash]<default: {}>

# by variable
ui_glyph name, options, html_options

# or by block
ui_glyph options, html_options do
  content
end

# or by hash
ui_glyph options

Options

The specific options for this component are:
  • status[symbol](:primary, :secondary, :success, :danger, :warning, :info, :light, :dark)(use component status method)
  • state[symbol](:disabled, :active)(use component state method)
  • size[symbol](:lg, :md, :sm)(use component size method)
  • flip[symbol](:horizontal, :vertical)
  • name[string]
  • rotate[number]
  • stack[integer]
  • label[string]
  • style[symbol]
  • cache[string](use to cache your component)
  • popover[string/hash](use component popover method)
  • controller[string](use component stimulus-options method)
  • action[string](use component stimulus-options method)
  • target[string](use component stimulus-options method)

Examples

Some examples explain how to use the options present in the component.

Glyph

  • Ruby
  • Html
ui_glyph "gem"

<i class="glyph fa fa-gem"></i>

Size

The size can be be defined through the `size` option that has the following arguments :

  • :lg
  • :md
  • :sm
  • 1
  • 2
  • 3
  • 4
  • 5




  • Ruby
  • Html
ui_glyph "gem", size: :lg
ui_glyph "gem", size: :md
ui_glyph "gem", size: :sm
ui_glyph "gem", size: 1
ui_glyph "gem", size: 2
ui_glyph "gem", size: 3
ui_glyph "gem", size: 4
ui_glyph "gem", size: 5

<i class="glyph fa fa-gem fa-1x"></i>
<i class="glyph fa fa-gem fa-2x"></i>
<i class="glyph fa fa-gem fa-3x"></i>
<i class="glyph fa fa-gem fa-5x"></i>

Type

  • fw : fixed width icon
  • Ruby
  • Html
ui_glyph "gem", type: "fw"

<i class="glyph fa fa-gem fa-fw"></i>

Label

Style

  • :solid
  • :regular
  • :light
  • :brands
  • Ruby
  • Html

Hash

You can create a ui_glyph with only one hash.

gem
  • Ruby
  • Html
ui_glyph { name: "gem", size: :sm, label: "gem" }

<i class="glyph fa fa-gem fa-1x"></i> gem

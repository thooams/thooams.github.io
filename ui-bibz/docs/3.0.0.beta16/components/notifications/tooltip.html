Tooltip

UiBibz::Ui::Core::Notifications::Tooltip

Documentation and examples for adding custom Bootstrap tooltips with CSS and JavaScript using CSS3 for animations and data-bs-attributes for local title storage

Usage

This component is an extension of component element. A Ui Bibz component consists of 3 arguments:
  • content[value/block]
  • html_options[hash]<default: {}>
  • options[hash]<default: {}>

 # by variable 
 tooltip content, options, html_options 

 # or by block 
 tooltip options, html_options do 
   content 
 end

Options

The specific options for this component are:
  • animation[Boolean]Apply a CSS fade transition to the tooltip<default: true>
  • boundary[String]Overflow constraint boundary of the tooltip
  • container[String/Element/Boolean]Delay showing and hiding the tooltip (ms) - does not apply to manual trigger type<default: false>
  • customClass[String]Add classes to the tooltip when it is shown
  • delai[Integer/Object]Appends the tooltip to a specific element<default: 0>
  • fallbackPlacement[Array]Allow to specify which position Popper will use on fallback
  • html[Boolean]Allow HTML in the tooltip.<default: false>
  • placement[String]Alias of position
  • popperConfig[Object]To change Bootstrap's default Popper config
  • position[String]How to position the tooltip - auto | top | bottom | left | right. When auto is specified, it will dynamically reorient the tooltip.<default: top>
  • template[String]Base HTML to use when creating the tooltip
  • title[String]Default title value if title attribute isn't present.
  • trigger[String]How tooltip is triggered - click | hover | focus | manual. You may pass multiple triggers; separate them with a space.<default: hover focus>

Note that for security reasons the sanitize, sanitizeFn, and allowList options cannot be supplied using data attributes.

Examples

Some examples explain how to use the options present in the component.

Options by Object

  • Ruby
  • Html
tooltip = UiBibz::Ui::Core::Notifications::Tooltip.new('My tooltip', position: :right)
ui_glyph "gem", tooltip: tooltip

<i data-bs-toggle="tooltip" data-bs-title="My tooltip" data-bs-placement="right" class="glyph fas fa-gem" data-bs-original-title="" title=""></i>

Options by Hash

  • Ruby
  • Html
ui_button "My button", tooltip: { title: 'My tooltip', position: :right}, status: :primary

<button data-bs-toggle="tooltip" data-bs-title="My tooltip" data-bs-placement="right" class="btn-primary btn" data-bs-original-title="" title="">My button</button>

Options by Boolean

To replace title notification by tooltip notification

If the options text: false and tooltip: true are present, the tooltip will take the content of the component.

My badge Share link
  • Ruby
  • Html
ui_badge "My badge", { type: :pill, tooltip: true, status: :danger }, { title: 'My tooltip' }
ui_button_link "Share link", glyph: 'share', tooltip: true, text: false

<span data-bs-toggle="tooltip" data-bs-title="My tooltip" class="bg-danger badge rounded-pill" data-bs-original-title="" title="">My badge</span>
<a data-bs-toggle="tooltip" data-bs-title="Share link" class="btn-secondary btn" role="button" href="#" data-bs-original-title="" title=""><i class="glyph fas fa-share"></i> <span class="visually-hidden">Share link</span></a>

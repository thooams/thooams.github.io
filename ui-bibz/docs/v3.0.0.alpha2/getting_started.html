Getting Started

Install

Add these lines in your Gemfile:
  • Code
# Gemfile
gem "ui_bibz", '~> 3.0.0.alpha2'
Run command:
  • Terminal
bundle install
yarn add ui-bibz-js

Assets with webpack

Put this line in: app/javascript/packs/applications.js
  • Code
# app/javascript/packs/applications.js
...

import UiBibzJs from "ui-bibz-js"
let uiBibzJs = new UiBibzJs
uiBibzJs.start()

...
Put this line in: app/javascript/stylesheets/application.sass
  • Code
# app/javascript/stylesheets/application.sass
...
@import "~ui-bibz-js/sass/ui-bibz"
...

Compatibility

Gem is compatible with Ruby version 2.x.x and Ruby On Rails 6.x.x.

Simple form

Gem is compatible with simple_form. All inputs fields can be used with this gem.

Build your own interface without writting HTML

Design through many components your own web interface thanks to Ui Bibz.
Ui Bibz is based on Bootstrap v5.0.0.alpha2 and Ruby on Rails 6.x.x.
No need to write html or update your code anymore.

A Ruby on Rails gem to facilitate the integration of your own interface. Compatible with Simple Form and Will Paginate.
